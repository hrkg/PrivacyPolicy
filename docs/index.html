プライバシーポリシー

📋はじめに

本プライバシーポリシーは、鍵メモ（以下「本アプリ」）がユーザーの個人情報をどのように取り扱うかについて説明するものです。

重要： 本アプリは、ユーザーのプライバシーを最優先に考えて設計されており、個人情報の収集は一切行いません。

🚫情報の収集について

収集しない情報

本アプリは以下の情報を一切収集いたしません：

  • 個人を特定できる情報（氏名、住所、電話番号、メールアドレス等）
  • 位置情報
  • デバイス識別情報
  • 使用状況や行動履歴
  • メモの内容や個人データ
  • アプリの利用統計

💾データの保存について

ローカル保存

本アプリで作成されたメモは、以下の方針で管理されます：

  • 端末内保存： すべてのメモデータは、ユーザーの端末内にのみ保存されます
  • 暗号化保護： パスコードは暗号化されて保存されます
  • 外部送信なし： メモデータが外部サーバーに送信されることはありません
  • 自動バックアップなし： iCloudやその他のクラウドサービスへの自動バックアップは行いません

🔒セキュリティ機能

プライバシー保護機能

  • パスコード保護： ユーザー設定の4桁パスコードでアプリ全体を保護
  • 生体認証： Face ID / Touch IDによる安全なアクセス制御
  • 自動ロック： アプリ切り替え時の自動ロック機能
  • データ暗号化： 端末内でのデータ暗号化保存

🌐ネットワーク通信について

インターネット接続不要： 本アプリは、一切のインターネット通信を行いません。そのため：

  • オフライン環境でも完全に動作します
  • データが外部に送信されるリスクがありません
  • 第三者による情報取得は不可能です
  • 広告表示やトラッキングは一切ありません

🤝第三者への情報提供

本アプリは、以下の理由により第三者への情報提供は一切ありません：

  • そもそも個人情報を収集していない
  • 広告ネットワークとの連携なし
  • 分析ツールの使用なし
  • 外部サービスとの連携なし

🔧ユーザーの権利

データの管理

ユーザーは以下の権利を有します：

  • データの削除： アプリを削除することで、すべてのデータを完全に削除できます
  • パスコードの変更： いつでもパスコードを変更または削除できます
  • メモの管理： 個別のメモを自由に編集・削除できます

👶未成年者の利用について

本アプリは年齢制限なく利用できますが、13歳未満のお子様が利用される場合は、保護者の方の同意のもとでご利用ください。なお、本アプリは個人情報を収集しないため、お子様のプライバシーに関するリスクはありません。

📱対応OS・デバイス

本プライバシーポリシーは、以下の環境で動作する本アプリに適用されます：

  • iOS 14.0以降を搭載したiPhone
  • iPadOS 14.0以降を搭載したiPad

🔄プライバシーポリシーの変更

本プライバシーポリシーは、法令の変更やアプリの機能向上に伴い、必要に応じて更新される場合があります。重要な変更がある場合は、アプリ内での通知またはApp Storeでの更新情報にてお知らせいたします。
